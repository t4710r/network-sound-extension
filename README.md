# network sound browser extension

an early iteration of a browser extension that makes little pinging noises whenever a website sends or receives a network request. this was created by a multidisciplinary team studying [machine langage](https://sfpc.study/sessions/winter-26/machine-language) at the [school for poetic computation](https://sfpc.study/)

each ping represents a network request or response by the site you're viewing. outgoing requests use a higher frequency and incoming requests sound lower. some sites are loud. some sites are quiet. why?
